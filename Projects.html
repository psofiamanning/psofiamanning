Projects

  • Pacmen

    • In this project a Pacman on demand is created by a click to the button "Add Pacman". With each click a Pacman appears on the screen at a random position. There is a "Start Game" button that as soon as it is clicked, it moves the Pacmen at random velocities. When one of the Pacmen hits an edge of the screen it changes direction and starts moving again. This exercise includes looping items through an array, creating and getting elements.

  • Eyes

    • In this project two eyes are created on the screen. User actions are binded by creating an event. The event is triggered when the cursor moves, making the created eyes move. The name of the event is ".onmousemove". To make the eyes, 3 classes are created: "eyes", "eye" and "ball". They are then called in the javascript file by using .getElementsByClassName. Apart from the js file, the repository also includes an index.html and styles.css file.
